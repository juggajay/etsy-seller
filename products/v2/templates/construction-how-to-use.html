How to Use Your Construction Invoice

Quick setup guide for builders and contractors

1 Getting Started
  • This PDF has two pages: an Invoice (page 2) and a Quote (page 3). Use whichever you need.
  • Open in Microsoft Edge, Adobe Acrobat Reader (free), or Chrome. All fields and auto-calculations work across viewers.
  • Click on any highlighted field to start typing. Blue-highlighted fields are editable; grey fields calculate automatically.
  • To add your logo: in Adobe Acrobat, go to Edit PDF → Add Image and position over the "Your Logo" area.
2 Filling In Your Invoice
  • Project Details: Enter the job site address, project reference number, and your ABN or license number at the top.
  • Labour Section: Enter each task description, hours worked, and hourly rate. The Amount column auto-calculates (hours × rate).
  • Materials Section: Enter materials used, quantity, and unit cost. Amount auto-calculates (qty × unit cost).
  • GST: Default is 10%. Change the rate if needed — GST amount recalculates automatically.
  • Retention: Default is 10%. The retention holdback amount is auto-calculated and deducted from the total.
  • Progress Claim: Track cumulative project progress. Enter total contract value and percentage complete. "Value to Date" and "This Claim" calculate automatically.
3 Using the Quote Page
  • Page 3 is a matching Quote template with the same labour and materials layout.
  • Send professional quotes and invoices that match — builds trust with clients.
  • Quote totals include GST but no retention or progress claim (those apply to invoices only).
  • Set the validity period (default: 30 days) and describe the scope of work in the footer.
Construction Tips

Save a blank copy: Always keep an unedited version as your master template.
Progress claims: Use for multi-stage projects — track cumulative % complete across invoices.
Retention: Standard 10% holdback. Set to 0% if not applicable to your contract.
File → Save As: Save each completed invoice with a unique name (e.g., INV-001-ClientName.pdf).
