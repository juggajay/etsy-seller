How to Use Your {{docType}} Template

Quick setup guide — you'll be ready in minutes

1 Using the PDF Template (Fillable)
  • Open the PDF file in any modern PDF reader — works great in Microsoft Edge, Adobe Acrobat Reader (free), and Chrome. All fields and auto-calculations work across viewers.
  • Click on any highlighted field to start typing — your business name, client details, line items, and more.
  • The Amount column calculates automatically when you enter Qty and Rate. Subtotal, Tax, and Total update instantly.
  • To add your logo: in Adobe Acrobat, go to Edit PDF → Add Image, then position it over the "Your Logo" area. Or leave it blank for a clean, minimal look.
  • When finished, go to File → Save As to save your completed {{docTypeLC}}. Print or email as needed.
2 Using the Word Template (.docx)
  • Open the .docx file in Microsoft Word or Google Docs (upload via Google Drive).
  • Click on any placeholder text (e.g., "Your Company Name") and type your information.
  • To add your logo, go to Insert → Pictures and place it over the "Your Logo" area.
  • Customise fonts, colours, or layout to match your brand — full editing freedom.
  • For the table, manually enter amounts. Word does not auto-calculate like the PDF version.
  • Save, export to PDF if needed, and share with your client.
3 Using the Excel Template (.xlsx)
  • Open in Microsoft Excel or upload to Google Sheets for cloud editing.
  • All calculation formulas are built in — enter Qty and Rate, and totals update automatically.
  • Great for tracking multiple {{docTypeLC}}s — duplicate the sheet tab for each new one.
  • Customise colours and formatting using Excel's built-in cell styling tools.
  • Export to PDF via File → Export → PDF for a professional finish.
Helpful Tips

Best for auto-calculate: Use the PDF (Adobe Acrobat Reader) or Excel version — both calculate totals automatically.
Best for customisation: Use the Word version — full control over design and layout.
Best for cloud access: Upload the Excel file to Google Sheets for easy sharing and collaboration.
Save a blank copy: Always keep an unedited version as your master template.
