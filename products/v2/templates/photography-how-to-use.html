How to Use Your Photography Invoice

Quick setup guide for photographers and studios

1 Getting Started
  • This PDF has two pages: an Invoice (page 2) and a Quote (page 3). Use whichever you need.
  • Open in Microsoft Edge, Adobe Acrobat Reader (free), or Chrome. All fields and auto-calculations work across viewers.
  • Click on any highlighted field to start typing. Blue-highlighted fields are editable; grey fields calculate automatically.
  • To add your logo: in Adobe Acrobat, go to Edit PDF → Add Image and position over the “Your Logo” area.
2 Filling In Your Invoice
  • Session Details: Enter the session type (wedding, portrait, event), date, and location.
  • Package: Describe your package, number of edited photos included, and the package price.
  • Add-ons: Extra photos, second photographer, prints, albums, travel fees. Qty × Rate auto-calculates each amount.
  • Tax: Default is 10%. Change the rate if needed — tax amount recalculates automatically.
  • Deposit: Enter the deposit amount and date paid. Balance Due auto-calculates (total − deposit).
  • Usage Rights: Specify print release or licensing terms for the client.
  • Delivery Timeline: Set expected turnaround for edited images (e.g., 4–6 weeks).
3 Using the Quote Page
  • Page 3 is a matching Quote template for booking confirmations and proposals.
  • Present your package and add-ons so clients see the full value before committing.
  • Quote totals include tax. No deposit fields (those apply to invoices only).
  • Set the validity period (default: 30 days) and include usage rights and delivery timeline.
Photographer Tips

Save a blank copy: Always keep an unedited version as your master template.
Deposits: Request 50% upfront to secure the booking date. Balance due 7 days before the session.
Usage rights: Always specify whether the client receives a print release or if licensing is separate.
File → Save As: Save each completed invoice with a unique name (e.g., INV-001-Smith-Wedding.pdf).
